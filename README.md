# Analogous-Exploration-for-CipherText-Visuals-in-Unsecured-Cloud-Computing
This project focuses on securing visual data stored in unsecured cloud environments using encryption-based techniques. The system protects sensitive image data by converting it into encrypted ciphertext format while still enabling controlled search functionality without exposing original content. 

Problem Statement:
Cloud systems are prone to the following problems:
Unauthenticated access
Data eavesdropping
Internal attacks
Data privacy breaches
Conventional encryption methods secure data but do not support the functionality of secure searching over encrypted images. This project proposes to address this issue.

Objectives:
Secure image data prior to uploading to cloud storage
Use AES encryption for confidentiality
Use SHA-256 hashing for integrity
Support searchable ciphertext functionality
Prevent unauthenticated access to confidential visual data

Technologies Used:
Java
AES Encryption
SHA-256 Hashing
MySQL Database
Cloud Simulation Environment

Key Features:
Encryption of images prior to cloud storage
Secure key generation system
Hash integrity verification
Ciphertext-controlled search
Secure cloud storage simulation
Security Controls Implemented
Administrative Controls:
Data access strategy development
Key management strategy development
Technical Controls:
AES encryption
Hash integrity verification
Secure authentication system
